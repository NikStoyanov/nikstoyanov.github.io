This blog is build with Hugo on Netlify. It also has a daily CRON job on Travis-CI which runs an update script to pull new running and swimming activities. Gym and random entries are manual since they can be unique and that prevents automation. The idea and majority of code is from Daniel Doubrovkine. The rest has been adapted to suit my purposes and you can fork the code on Github. The Ruby Strava API client is used to pull new activities and post process them to markdown blogposts.
